# 3D Neon Lightbulb

Interactive 3D visualization of a neon lightbulb using Three.js.

![Neon Lightbulb Preview](https://img.shields.io/badge/Three.js-r128-orange)
![HTML5](https://img.shields.io/badge/HTML5-Canvas-blue)

## Features

- 🏠 **Realistic Design** - Orange cord, black metallic fixture base, and neon bulb
- ✨ **Glowing Effects** - Multiple glow layers with additive blending
- 🔘 **Interactive Switch** - Click the wall switch to toggle light on/off
- 💡 **Pulsating Glow** - Subtle breathing animation when lit
- ⚡ **Flicker Effect** - Random neon flicker simulation
- 🎮 **Orbit Controls** - Rotate and zoom the view

## Demo

Open `neon-lightbulb.html` in any modern browser.

### Live Demo
Visit: [https://antono4.github.io/3D-Neon-Lightbulb/neon-lightbulb.html](https://antono4.github.io/3D-Neon-Lightbulb/neon-lightbulb.html)

## Controls

| Action | Description |
|--------|-------------|
| Click Switch | Toggle light on/off |
| Drag | Rotate camera view |
| Scroll | Zoom in/out |

## Technologies

- **Three.js r128** - 3D rendering
- **OrbitControls** - Camera interaction
- **WebGL** - Hardware-accelerated graphics

## Installation

No installation required. Simply open `neon-lightbulb.html` in a browser.

### Local Development

```bash
# Clone the repository
git clone https://github.com/antono4/3D-Neon-Lightbulb.git

# Open in browser
open neon-lightbulb.html
# or
python3 -m http.server 8080
# Then visit http://localhost:8080
```
